# Spring Core Examples

## Overview
This repository contains basic examples to help you understand some fundamental **Spring Core** concepts, including:

- Dependency Injection (Constructor, Setter, Field)
- Bean Scopes (Singleton, Prototype, Request, Session)
- Bean Lifecycle (@PostConstruct, @PreDestroy)
- Using @Component and @Autowired

These examples are meant for beginners to get hands-on experience with Spring Framework core features.

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/spring-core-examples.git
````

2. Open the project in your IDE.
3. Run the example classes to see Spring Core in action.

## Goal

To provide a simple and practical way to learn and experiment with Spring Core concepts.
